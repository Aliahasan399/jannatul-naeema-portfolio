Data Deletion Instructions

Last updated: June 03, 2026

How to Request Data Deletion

If you wish to delete your data associated with this Facebook Page management application, please follow these steps:

Method 1: Email Request

Send an email to mdmasum399222@gmail.com with the subject line "Data Deletion Request" including:

  • Your Facebook user ID
  • The Page ID (if applicable)
  • A brief description of the data you want deleted

We will process your request within 7 business days.

Method 2: Facebook App Removal

You can also remove the app entirely from your Facebook account:

  1. Go to Facebook Settings
  2. Click "Apps and Websites"
  3. Find this app and click "Remove"
  4. This will revoke all permissions and delete associated data

Data Deletion Callback

This application does not permanently store user data. All data is accessed in real-time through the Facebook Graph API and is not retained on our servers.
